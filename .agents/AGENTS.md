# AI Agent Rules

These rules apply to all AI agents interacting with this workspace. You MUST follow them strictly.

## ESLint Requirement
Whenever you modify any TypeScript (.ts) or JavaScript (.js) files in this project, you MUST immediately run `npm run lint` or `npm run lint:fix` to ensure code quality and code styling guidelines are met. Do not ask for permission, just run it proactively after completing your modifications.
